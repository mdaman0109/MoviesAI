This is an MoviesAI app which uses Chatgpt API to suggest movies based on our prompts
It is based on React JS, Firebase and Tailwind CSS

VITE + Tailwind CSS

Login/Sign up page
    Login/SignUp Form
    Redirect to Browse
Browse - After Auth
    Header
    Main Movies
        Trailer in background
        Title & Description
        Movie Suggestions
        Movies List
MoviesAI
    SearchBar which gives movie suggestions
